## How to use?
1. create .env file
2. add your database credentials to .env file

### Example .env file
```
DB_PASS=
DB_USER=
JWT_SECRET=
```
3. run `npm install` or `yarn`
4. run `npm run dev` or `yarn dev`
5. open `http://localhost:3000` in your browser

## For GraphQL
- open `http://localhost:3000/graphql` in your browser
